# Nova5 Items Field

A Laravel Nova 5 custom field to manage repeatable non-empty string values via an intuitive interface.

## Features

- ✅ Drag-and-drop reordering
- ✅ Comma-paste splitter
- ✅ Nova 5 + Vue 3 Composition API support
- ✅ Fully typed PHP 8.2+
